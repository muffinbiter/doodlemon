# DOODLEMON
AI Drawing Monster Battle Game

Your drawings become monsters.

🎮 Play Demo  
https://muffinbiter.github.io/doodlemon/

---

## Overview

**DOODLEMON** is an experimental drawing-based monster game where the creatures players battle are generated directly from their sketches.

Instead of collecting monsters through random encounters, players draw their own creatures, and AI analyzes the drawing to determine its type, stats, and abilities.

The project demonstrates how **player-created content can be transformed into gameplay systems through AI interpretation**.

---

## Core Gameplay Loop

Explore stage → Encounter enemy →  
Draw a new Doodlemon → AI analyzes drawing →  
Generate monster → Turn-based battle.

---

## Key Features

**Drawing System**  
Players freely draw monsters on a canvas.

**AI Analysis Engine**  
Color, shape, and drawing density determine monster type and stats.

**Element System**  
Color influences elemental type such as Fire, Water, Grass, and Electric.

**Doodle Book Tiers**  
Different analysis levels unlock stronger monster abilities.

**Exploration Gameplay**  
Side-scrolling stages with coin collection and enemy encounters.

**Turn-Based Battles**  
Pokémon-inspired 1v1 battle system with elemental advantages.

---

## Tech Stack

HTML5 Canvas  
Vanilla JavaScript  
Pixel-level image analysis using Canvas ImageData API  
Single-file game architecture

---

## Project Background

DOODLEMON was built as a prototype exploring how **AI can transform user-generated drawings into meaningful gameplay elements**, creating an immediate feedback loop between creativity and game mechanics.
